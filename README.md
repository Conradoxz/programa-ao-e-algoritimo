# 💻 Programação e Algoritmos | Construindo a Lógica

Bem-vindo(a) ao repositório de **Programação e Algoritmos**! Este espaço foi criado para documentar toda a minha jornada desenvolvendo o raciocínio lógico e resolvendo problemas matemáticos e cotidianos através do código no 1º ano técnico da ETEC.

---

## 🛠️ Tecnologias & Ferramentas

No desenvolvimento da lógica e dos primeiros programas, utilizo:

* **Linguagens de Aprendizado:** Portugol (VisualG / Portugol Studio) ou linguagens de mercado como Python / Java (mantenha a que você usa).
* **Ambientes de Desenvolvimento:** VS Code, VisualG ou IDEs específicas da disciplina.
* **Paradigma:** Programação Estruturada e Sequencial.

---

## 🚀 Meus Focos de Estudo & Aplicação

* **Lógica de Programação:** Entendimento profundo de variáveis, constantes, operadores matemáticos e operadores lógicos.
* **Estruturas de Decisão:** Controle de fluxo em aplicações utilizando estruturas condicionais (`Se/Então/Senão` ou `if/else`).
* **Estruturas de Repetição:** Otimização de rotinas e tratamento de dados em massa usando loops (`Para`, `Enquanto`, `Faça-Enquanto`).
